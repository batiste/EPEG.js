Peacock language

A beautiful simple language that compile to JavaScript. The language doesn't try to be too smart and is mainly syntactic sugar around common JavasScript annoyances. Features:

  1. Multiline strings
  2. Significant whitespace
  3. Short function declaration and lambda expression
  4. Function parameters default values
  5. Simple for loop syntax
  6. Sensible scoping rules unlike CoffeeScript
  7. Classes (no inheritance yet)

More info on the github page https://github.com/batiste/EPEG.js
